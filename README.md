# Telecom Customer Churn Analysis & Prediction
### Internship Project - Saiket Systems

## Project Overview
This repository contains the work completed during my **Data Analysis Internship** at **Saiket Systems**. The project focuses on analyzing customer churn in a telecommunications company to identify at-risk customers and provide data-driven recommendations to improve retention.

## Tasks Performed

### 1. Data Cleaning & Preprocessing
- Identified data types and checked for missing values across 21 columns.
- Handled feature engineering by converting `TotalCharges` to numeric and processing categorical variables.
- Prepared the dataset for modeling by removing non-essential identifiers like `customerID`.

### 2. Exploratory Data Analysis (EDA)
- Conducted statistical summaries to understand the distribution of tenure and monthly charges.
- Created visualizations (Box plots, Violin plots, and Pair plots) to identify correlations between service types and churn.

### 3. Predictive Modeling
- Developed a churn prediction model using **Logistic Regression**.
- Evaluated model performance using metrics such as **Accuracy, Precision, Recall, and F1-score**.

### 4. Retention Strategies
- Calculated **Customer Lifetime Value (LTV)** to prioritize high-value retention.
- Identified key factors influencing churn (e.g., contract types and technical support).
